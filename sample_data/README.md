# Sample Data

These files are illustrative development fixtures for bootstrapping the local OperationalDecisionSupportSystemForWaterSecurity database.

- They are intentionally small.
- They are not authoritative geographic boundaries.
- They are not real epidemiological claims.

Use them only for local development and testing of the ingestion path.

Included fixtures:

- `admin_boundaries.csv`
- `district_week_labels.csv`
- `district_static_covariates.csv`
- `district_week_weather.csv`
